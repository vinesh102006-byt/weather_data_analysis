# Weather Data Analysis and Rainfall Prediction

## Project Overview

This project performs **Weather Data Analysis** and **Rainfall Prediction** using Python and Machine Learning.

The dataset is analyzed using data visualization techniques, and a predictive model is built using:
- Pandas for data handling
- Matplotlib & Seaborn for visualization
- Scikit-learn for Machine Learning

The project predicts rainfall based on:
- Minimum Temperature (`MinTemp`)
- Maximum Temperature (`MaxTemp`)

The machine learning algorithm used is Linear Regression.

---

# Features

- Load and analyze weather datasets
- Explore dataset statistics
- Handle missing values
- Create data visualizations
- Generate correlation heatmaps
- Build a rainfall prediction model
- Evaluate model performance using Mean Squared Error (MSE)
- Compare actual vs predicted rainfall values
